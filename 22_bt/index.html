This is a container

Containers are a key part of Bootstrap, and they are just what they sound like. They're there to contain and pad content.

Container type 0 - fills full width

This is a grid

Grids are just containers with any number of rows which can contain up to 12 columns each.
These columns also use colors defined by Bootstrap as root variables. A full list of CSS variables can be found here.
You can make divs span more than 1 column. This one is class="col-8", spanning 8 of the 12 columns.

This is a carousel

There's a custom duration for each slide, and the carousel resizes to fill the width of the current window. You can create a carousel by adding a div with class="carousel slide" containing divs with class="carousel-item"
